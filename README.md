# Config

Configuration files

## Usage

Move or import the configuration files

### Windows

#### Git

`.gitconfig` and `.gitignore`

```
%USERPROFILE%
```

#### Git Bash

`.minttyrc`

```
%USERPROFILE%
```

#### Sublime Text

`Preferences.sublime-settings`

```
%APPDATA%\Sublime Text\Packages\User
```

#### Visual Studio 2022

`Exported-2022-11-30.vssettings`

`Tools` > `Import and Export Settings...` > `Import selected environment settings`
